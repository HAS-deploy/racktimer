drop set — strength-training glossary

What it means

A drop set is a technique where you perform a set to or near failure, immediately reduce the weight, and continue repping without resting. You "drop" the load — typically by 10–30% — and keep going until you hit failure again. You can repeat this process for two, three, or more drops in a single continuous effort.

The goal is to extend a set beyond the point where you could continue at the original weight, increasing total volume and muscular fatigue in a short amount of time.

Why it matters

Drop sets come up most often when someone wants to maximize muscle fatigue at the end of a workout without adding another full working set. They're popular for hypertrophy-focused training because they push a muscle through multiple failure points in one go. You'll also see them used when time is limited — a single drop set can approximate the stimulus of two or three straight sets in a fraction of the time.

They're best suited to isolation or machine exercises where changing the load quickly is practical (like a cable stack or dumbbell rack). Using them on barbell squats, for example, is awkward and slows things down enough to undermine the point.

Example

You're finishing a dumbbell lateral raise set with 20 lb dumbbells and reach failure at 12 reps. You immediately grab the 12 lb dumbbells and bang out another 8 reps until you fail again. That's one drop. Some people drop a second time to 8 lb for a final 6–8 reps. The whole sequence counts as one drop set.

Common confusions

Drop set vs. superset: A superset moves between two different exercises with no rest. A drop set stays on the same exercise and only changes the load.

Drop set vs. rest-pause: A rest-pause set involves a brief pause (usually 10–20 seconds) before continuing with the same weight. A drop set requires no pause but reduces the weight instead.

"Failure" confusion: People sometimes stop a drop set too conservatively, leaving several reps in reserve before each drop. For a drop set to work as intended, each phase should reach or approach true muscular failure — otherwise you're just doing multiple light sets with sloppy transitions.

Overuse: Drop sets are a finishing tool, not a training method to build an entire session around. Because they generate high fatigue quickly, using them on every exercise in a session typically leads to poor performance across the board and inadequate recovery.
