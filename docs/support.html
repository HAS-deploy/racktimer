Common questions

How do I restore my purchase?

Tap Settings → Restore purchases. If you've bought the Lifetime Unlock on any Apple ID signed into this device, it will be restored.

Does RackTimer sync across devices?

No. RackTimer is local-first and stores everything on the device. The Lifetime Unlock is tied to your Apple ID, so purchases restore on any device signed into the same Apple ID.

I bought the unlock but it didn't apply.

Make sure you're signed into the same Apple ID you used to purchase, then tap Restore purchases. If it still doesn't work, email support.
